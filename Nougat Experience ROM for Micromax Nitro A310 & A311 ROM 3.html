Installation:-

1. Download ROM

2. Download Gapps

3. Download SuperSu.zip (Skip If You Don't Want Google Assistant and More Cool Features)

4. Place all on External Storage

5. Open TWRP

6. Make a backup of current ROM (Optional)

7. Go to Wipe-

    7.1.  Select Advance Wipe

    7.2.  Select System, Dalvik, Cache, Internal Storage

    7.3.  Wipe this selected options

8. After wipe go to install and choose External Storage

9. Browse ROM and select it

10. Add Gapps, Sound Patch and SuperSu (A310 Users can add A310 Patch)

11. Flash it.

12. Reboot it.

13. After mobile rebooted choose LAWNCHAIR as Default Launcher.

14. Go to Lawnchair Settings and go to UI > Icon Pack > Select Pixel icon pack.

   15. Enable Transparent Dock, Extract color from wallpaper, Enable Blur, Use white Google icon

   16. Set Number of Columns, Rows, Dock to 5

   17. Go to Main Settings > Pixel Style

   18. Enable Show Google Assistant Button, Use Round Apps Search bar

   19. Go to HOME and see Changes

20. Next is Xposed and its Module

    21. Go to menu and open Xposed app

    22. Install Xposed

    23. Go to Xposed module and install Assistant Enabler, Android N-ify, Gravity Box MM

    24. Set all this application and apply module and then Reboot.

25. Go to Settings > Apps > Google App and Clear Data

26. Press home button (Virtual button, if not visible then Go to GravityBox and set up) and Google Assistant Will Works.
