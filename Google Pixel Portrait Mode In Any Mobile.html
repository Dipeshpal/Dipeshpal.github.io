How To Install Stock ROM On Micromax Nitro

Flash pure stock ROM.
Unbrick guide for Canvas Nitro.
First of all, you need to download stock ROM.

Download stock ROM from here...

For A310 users: Download

For A311 users: Download

Download YGDP assembly from here: Download

Download drivers from here: Download

 

1. Install vcom install.exe and the installdrv64.exe from the vcom folder.

### For Windows 8 users: ###
1. Go to the windows charm bar/settings/change pc settings/general/scroll to advanced startup. It will reboot.
2. Press trouble shoot/advanced/startup../ restart special option../press 7 (disable driver..)
3. It will boot you normally to windows but now u can install unsigned drivers like Vcom.
4. Open device manager.
### Now windows 8.1 users. ###

1. Go to the windows charm bar/settings/change pc settings/Update and recovery/recovery/ scroll to advanced startup and click restart now...
2. Press troubleshoot/advanced/startup settings/ restart.
3. Press 7.
4. It will boot you normally to windows but now u can install unsigned drivers like Vcom.
5. Open device manager.
If your windows 7 or Windows XP users. Advanced startup ur pc. Then select disable driver signature enforcement.
### Now all users follow this... ###
1. Switch off the phone. Remove battery.
2. Connect the phone to the same usb you used when you installed the other drivers (coolpad , pdanet).
3. On device manager new you will see a new entry..
4. Double click it and press update driver.
5. Select to manually guide windows to the driver files ( Win7 folder inside vcom folder).
6. Go on with the installation , windows will warn you about unsigned drivers, press continue any way.
7. Done, you will see a new entry on device manager in the " android phone" tab.

Now you can flash ur device with ygdp or sp flash tool...

1.. Openygdp assembly. Enter password. 369
2. Click on config button.
3. Choose your phone (8297w_I01).
4. Select CPB file u downloaded.
5. apply yourselection.. Wait few minutes for your ROM unpack.
6. Click on start button.

Turn off your mobile. connect your mobile.
Firmware downloading starts automatically. It takes some time to finish.
